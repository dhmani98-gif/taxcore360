TAXCORE360 LOGO IMAGES
======================

Place your logo images here with the following names:

1. logo.png (REQUIRED)
   - Main horizontal logo
   - Used in: Sidebar, Landing Page (Navbar & Footer), Auth Screen
   - Recommended size: 400x100px or similar ratio
   - Format: PNG with transparent background
   - Height in UI: 40-50px (will be scaled)

2. logo-vertical.png (OPTIONAL)
   - Vertical/square version for mobile apps
   - Used in: Mobile app icon, profile pictures
   - Recommended size: 512x512px (square)
   - Format: PNG with transparent or white background

3. favicon.ico (OPTIONAL)
   - Browser tab icon
   - Size: 32x32px or 64x64px
   - Format: ICO or PNG

COLOR SPECIFICATIONS:
- Primary Blue: #1e40af (dark blue)
- Accent Blue: #3b82f6 (bright blue)
- Text: Dark blue for "TAXCORE", lighter blue for "360"

NOTE: The website is already configured to use /images/logo.png
Just replace the file and refresh the page to see the changes.
